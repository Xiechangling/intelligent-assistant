  <action>
重构 SessionHeader 组件 (lines 714-745)，简化为轻量级单行布局：

**新结构**:
```tsx
function SessionHeader({ header, mode }: { header: DesktopSessionHeader; mode: 'project' | 'conversation' }) {
  return (
    <div className="workspace__session-header">
      <div className="workspace__session-header-main">
        <div className="workspace__session-title-group">
          <span className="workspace__eyebrow">{mode === 'project' ? 'Attached session' : 'Conversation session'}</span>
          <h2>{header.title}</h2>
        </div>
        <StatusChip status={header.workflowStatus} />
      </div>
      {header.currentActivitySummary && (
        <p className="workspace__session-activity">{header.currentActivitySummary}</p>
      )}
    </div>
  )
}
```

**移除的元素**:
- `workspace__session-header-card` 类名（改为 `workspace__session-header`）
- `workspace__session-meta-grid` 整个元数据网格（workspace, model, last activity, session state）

**保留的元素**:
- Eyebrow 标签（"Attached session" / "Conversation session"）
- Session title (h2)
- StatusChip（显示 workflowStatus）
- Current activity summary（如果存在）

**理由**: 元数据网格中的信息在其他位置已有展示：
- Workspace → 顶栏面包屑 + 左侧栏
- Model → 顶栏模型选择器
- Last activity → 左侧栏会话列表
- Session state → StatusChip 已显示

保留 currentActivitySummary 因为它是动态的、与当前操作相关的上下文。
