  <files>src/app/hooks/useKeyboardShortcuts.ts</files>
  <action>
    创建全局快捷键管理 hook：
    
    **useKeyboardShortcuts.ts**:
    - 接收参数: shortcuts (Record<string, () => void>)
    - 使用 useEffect 注册全局 keydown 事件监听器
    - 检查 event.target 是否为 INPUT/TEXTAREA（避免输入框冲突，参考 09-RESEARCH.md Pitfall 1）
    - 支持组合键检测：
      - Ctrl+F: 打开搜索
      - Ctrl+N: 新建会话
      - Ctrl+1/2/3: 切换模式
      - Alt+Left/Right: 前进后退
      - Escape: 关闭模态框
    - 使用 (e.metaKey || e.ctrlKey) 兼容 Mac/Windows
    - 清理函数：removeEventListener
    
    **示例用法**:
    ```typescript
    useKeyboardShortcuts({
      'ctrl+f': () => setSearchOpen(true),
      'escape': () => setSearchOpen(false)
    })
    ```
    
    参考 09-RESEARCH.md Pattern 1 和 Code Examples 中的实现。
