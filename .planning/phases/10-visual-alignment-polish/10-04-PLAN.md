**当前状态（需要修改）：**
- AttachmentList: `transition: all 0.2s ease` → `transition: all 200ms ease-out`
- CustomizeMenu: `transition: background-color 150ms ease` → `transition: background-color 200ms ease-out`
- GlobalSearch: `transition: background-color 150ms ease` → `transition: background-color 200ms ease-out`
- ModeTabs: `transition: all 0.2s ease` → `transition: all 200ms ease-out`
- NavigationButtons: `transition: background-color 0.2s ease-out` → `transition: background-color 200ms ease-out`（已正确，仅统一单位）
- SidebarTopActions: `transition: background 0.2s` → `transition: background 200ms ease-out`
- SidebarTopActions: `transition: all 0.2s` → `transition: all 200ms ease-out`
- VoiceInput: `transition: all 0.2s ease` → `transition: all 200ms ease-out`
- WindowControls: `transition: background-color 0.2s ease-out` → `transition: background-color 200ms ease-out`（已正确，仅统一单位）

**统一规则：**
- 时长：200ms（不使用 0.2s）
- 缓动函数：ease-out
- 属性：根据实际需要选择 `all` 或具体属性（如 `background-color`, `opacity`, `transform`）

**最佳实践：**
- 背景色变化：`transition: background-color 200ms ease-out`
- 透明度变化：`transition: opacity 200ms ease-out`
- 变换动画：`transition: transform 200ms ease-out`
- 多属性变化：`transition: all 200ms ease-out`（谨慎使用，可能影响性能）

逐个文件修改，确保所有 transition 声明都符合规范。
