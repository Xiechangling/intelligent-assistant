  <action>
更新 `[data-theme='light']` 选择器中的 CSS 变量，对齐官方浅色主题：

**背景色优化（接近白色）：**
- `--bg-app`: #ffffff（纯白应用背景）
- `--bg-shell`: #ffffff（纯白外壳）
- `--bg-sidebar`: #fefefe（极浅灰侧栏）
- `--bg-canvas`: #ffffff（纯白画布）
- `--bg-panel`: #fafafa（浅灰面板）
- `--bg-elevated`: #ffffff（纯白提升层）
- `--bg-muted`: #f5f5f5（柔和灰）
- `--bg-soft`: #fefefe（柔和背景）

**分隔线优化（柔和不刺眼）：**
- `--border-subtle`: rgba(0, 0, 0, 0.06)（极柔和分隔线）
- `--border-strong`: rgba(0, 0, 0, 0.12)（强调分隔线）
- `--border-highlight`: rgba(0, 0, 0, 0.08)（高亮分隔线）

**文本色优化（保持可读性）：**
- `--text-primary`: #1a1a1a（主文本，接近黑色）
- `--text-secondary`: #525252（次要文本，中灰）
- `--text-tertiary`: #737373（三级文本，浅灰）

保持其他变量不变（accent、warning、danger 等）。
  </action>
