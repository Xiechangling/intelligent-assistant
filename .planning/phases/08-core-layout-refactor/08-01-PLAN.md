<task type="auto">
  <name>Task 2: 添加导航历史状态和 actions</name>
  <files>src/app/state/appShellStore.ts</files>
  <action>
在 AppShellState 接口中添加导航历史字段：
```typescript
navigationHistory: string[]  // Session IDs
navigationIndex: number      // Current position in history (-1 = no history)
```

添加导航 actions：
```typescript
goBack: () => void
goForward: () => void
pushNavigation: (sessionId: string) => void
```

实现逻辑（参考 RESEARCH.md Pattern 2）：
- goBack: 如果 navigationIndex > 0，减少 index，调用 resumeSession
- goForward: 如果 navigationIndex < history.length - 1，增加 index，调用 resumeSession
- pushNavigation: 添加 sessionId 到 history，截断 index 之后的历史，限制最多 50 条

在 resumeSession 中调用 pushNavigation 记录导航。
  </action>

  </verify>
