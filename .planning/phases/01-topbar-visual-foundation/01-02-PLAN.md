**移除的状态和函数：**
- `mode`, `activeSession`, `credentialStatus`, `activeSessionModelOverride`, `getDesktopWorkflow` — 这些信息移至 session header（Phase 2）
- `handleProjectPick` — 项目选择移至左侧栏（Phase 3）
- `openInspector('context')` — Context 视图合并到 Settings

**新的 JSX 结构：**
```tsx
export function TopToolbar() {
  const {
    activeProjectPath,
    globalDefaultModel,
    setGlobalDefaultModel,
    setRightPanelOpen,
    setRightPanelView,
  } = useAppShellStore()

  const handleModelChange = (event: ChangeEvent<HTMLSelectElement>) => {
    const nextModel = event.target.value as ModelId
    setGlobalDefaultModel(nextModel)
  }

  const openSettings = () => {
    setRightPanelView('settings')
    setRightPanelOpen(true)
  }

  // 简化项目名称显示（仅显示最后一段路径）
  const projectName = activeProjectPath 
    ? activeProjectPath.split(/[\\/]/).filter(Boolean).pop() || 'Project'
    : 'No workspace'

  return (
    <div className="toolbar">
      {/* 左侧：面包屑（项目名称） */}
      <div className="toolbar__breadcrumb">
        <span className="toolbar__project-badge">{projectName}</span>
      </div>

      {/* 中央：模型选择器 */}
      <div className="toolbar__center">
        <select 
          className="toolbar__model-select" 
          value={globalDefaultModel} 
          onChange={handleModelChange}
          aria-label="Model selector"
        >
          <option value="claude-opus">claude-opus</option>
          <option value="claude-sonnet">claude-sonnet</option>
          <option value="claude-haiku">claude-haiku</option>
        </select>
      </div>

      {/* 右侧：设置入口 */}
      <div className="toolbar__actions">
        <button 
          className="toolbar__icon-button" 
          onClick={openSettings} 
          title="Settings"
        >
          <Settings size={15} />
        </button>
      </div>
    </div>
  )
}
```

**为什么这样做：**
1. **移除项目选择器按钮** — 项目切换应在左侧栏完成（更符合导航逻辑）
2. **移除会话上下文栈** — 会话信息应在中央工作区 session header 显示（更贴近上下文）
3. **移除状态芯片** — 状态应在 session header 或内联卡片显示（更贴近相关操作）
4. **合并 Context 按钮** — Context 和 Settings 都是辅助信息，合并为单个设置入口

**保留桌面 GUI 优势：** 保留圆角按钮、阴影、hover 动画（REQ-07）。
