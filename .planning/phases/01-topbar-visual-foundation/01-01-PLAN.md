创建新文件 `src/app/components/KeyboardShortcutHint.tsx`，实现快捷键提示组件：

**接口定义：**
```typescript
interface KeyboardShortcutHintProps {
  shortcut: string;        // 如 "ctrl+t", "alt+p", "esc"
  action?: string;         // 可选动作描述，如 "to open"
  parens?: boolean;        // 是否用括号包裹，默认 false
}
```

**实现要点：**
1. 检测操作系统：`window.navigator.platform.toLowerCase().includes('mac')`
2. 映射快捷键符号：
   - macOS: `ctrl` → `⌘`, `alt` → `⌥`, `shift` → `⇧`
   - Windows/Linux: `ctrl` → `Ctrl`, `alt` → `Alt`, `shift` → `Shift`
3. 首字母大写：`ctrl+t` → `Ctrl+T` 或 `⌘T`
4. 格式化输出：`{shortcut} {action}` 或 `({shortcut} {action})`

**样式：**
```tsx
<span className="keyboard-hint">
  <kbd className="keyboard-hint__key">{formattedShortcut}</kbd>
  {action && <span className="keyboard-hint__action"> {action}</span>}
</span>
```

**CSS 样式（添加到 app-shell.css 末尾）：**
```css
.keyboard-hint {

  align-items: center;
  gap: 4px;
  opacity: 0.7;
  font-size: 12px;
}

.keyboard-hint__key {
  font-family: inherit;

  background: rgba(255, 255, 255, 0.06);
  border: 1px solid rgba(255, 255, 255, 0.08);
}

  opacity: 0.8;

```

**参考：** INTERACTION-MODEL.md 第 276-291 行，官方快捷键提示组件实现。
