   - When expanded: opacity 1, pointer-events auto

4. Add proper z-index layering:
   - `.bottom-panel`: z-index: 10 (above main content)
   - `.app-shell__drawer`: z-index: 20 (above bottom panel)
   - `.app-shell__top`: z-index: 30 (above all)

   - Add active state with pressed effect
   - Ensure ChevronUp/ChevronDown icon rotates smoothly

```css
.bottom-panel {
  max-height: 400px;
  overflow: hidden;
  transition: max-height 200ms ease-out;
  z-index: 10;
  position: relative;

.bottom-panel[data-collapsed="true"] {
  max-height: 48px;
}

.bottom-panel__content {
  opacity: 1;
  pointer-events: auto;
  transition: opacity 160ms ease;
```
