## 应用启动方式

### 开发模式启动

**方式 1：前端 + 后端同时启动（推荐）**

```bash
# 安装依赖（首次运行）
npm install

# 启动开发服务器（前端 + Tauri 后端）
npm run tauri dev
```

这会同时启动：
- Vite 开发服务器（前端热重载）
- Tauri 应用窗口（原生桌面应用）

**方式 2：仅前端开发**

```bash
# 启动 Vite 开发服务器
npm run dev
```

然后在浏览器访问：`http://localhost:5173`

**注意：** 此模式下 Tauri 后端功能不可用（如文件系统、原生对话框等）

### 运行测试

**运行所有 E2E 测试：**

```bash
npx playwright test
```

**运行特定测试套件：**

```bash
# 启动测试
npx playwright test tests/e2e/startup.spec.ts

# 审批流程测试
npx playwright test tests/e2e/approval-flow.spec.ts

# 主题切换测试
npx playwright test tests/e2e/theme-switching.spec.ts

# 键盘导航测试
npx playwright test tests/e2e/keyboard-navigation.spec.ts

# 抽屉面板测试
npx playwright test tests/e2e/drawer-panel.spec.ts
```

**以 UI 模式运行（可视化调试）：**

```bash
npx playwright test --ui
```

**查看测试报告：**

```bash
npx playwright show-report
```
